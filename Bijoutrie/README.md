# Bijouterie - Catalogue PHP

Un site de catalogue de bijouterie moderne et responsive construit avec PHP et Tailwind CSS.

## ?? Installation locale

### Prérequis
- PHP 7.4+
- Un serveur web (Apache, Nginx) ou le serveur PHP intégré

### Étapes d'installation

1. **Cloner le dépôt**
```bash
git clone https://github.com/K0ZAHAAAAAA1232342/GitBook.git
cd Bijoutrie
```

2. **Démarrer un serveur PHP local**
```bash
php -S localhost:8000
```

3. **Ouvrir dans le navigateur**
```
http://localhost:8000
```
